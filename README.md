## Planned Features

- running tests with CLI

## Development

### Prerequisites

You will need

- [Node JS](https://nodejs.org/en/)

### Installing

```sh
git clone git@github.com:dgoldstein1/swagger-integration-test-UI.git
cd swagger-integration-test-UI
npm install
npm start
```
